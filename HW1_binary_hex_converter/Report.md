# 作業報告 - Binary Hex Converter

## 一、設計概念
本程式使用除法與乘法方式實作進位轉換：
- 十進位 → 二進位：除2取餘數
- 十進位 → 十六進位：除16取餘數
- 反向：乘基底累加

## 二、功能
- Binary ↔ Decimal ↔ Hex
- GUI 操作
- 錯誤輸入檢查

## 三、測試案例
| 輸入 | 輸出 |
|------|------|
| 1010 | 10 |
| 255 | FF |
| FF | 255 |

## 四、結果
成功完成三進位轉換且無使用內建函式。
