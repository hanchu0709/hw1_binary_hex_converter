# 簡介
實現非使用內建函式，完成2進位(Binary)、10進位(Decimal)與16進位(Hexadecimal)之間的轉換，並提供圖形化使用者介面(GUI)

# AI使用
- 在圖形化介面(tkinter)部分，是完全由AI協助
- 三種進位之間的轉換，有讓AI幫忙進行優化
